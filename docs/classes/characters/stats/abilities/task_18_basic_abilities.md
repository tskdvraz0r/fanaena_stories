## 5. Атрибуты экземпляра класса;
*Атрибуты* экземпляра класса не предусмотрены.
Экземпляр класса должен иметь следующие *атрибуты*:
* _class_modifiers: BasicModifiers - экземпляр класса модификаторов характеристик персонажа;
* _class_proficiency_bonus: BasicProficiencyBonus- экземпляр класса бонуса мастерства персонажа;
* _class_posessions: BasicPosessions - экземпляр класса "владений" персонажа

* _athletics: int - (Сила) навык отвечает за следующие проверки: бег, прыжки, плавание и лазание;
* _acrobatics: int - (Ловкость) навык отвечает за следующие проверки: бег по льду, балансирование на натянутом канате или палубе корабля во время шторма;
* _stealth: int - (Ловкость) навык отвечает за следующие проверки: укрыться от врага, прокрасться мимо, незаметно подкрасться;
* _sleight_of_hand: int - (Ловкость) навык отвечает за следующие проверки: жонглирование, ловкие трюки (подбросить что-то кому-то, стащить что-то из кармана/пояса);
* _analysis: int - (Интеллект) навык отвечает за следующие проверки: поиск подсказок, выводы на основе подсказок, оценка окружения, поиск по подсказкам;
* _history: int - (Интеллект) навык отвечает за следующие проверки: знания исторических событий, легендарных личностей, древних королевств, былых спорах, недавних войнах и исчезнувших цивилизациях;
* _magic: int - (Интеллект) навык отвечает за следующие проверки: знания заклинаний, магических предметов, мистических символов, магических традиций;
* _nature: int - (Интеллект) навык отвечает за следующие проверки: знания о местности, о животных, о погоде и естественных циклах;
* _religion: int - (Интеллект) навык отвечает за следующие проверки: знания о божествах, ритуалах и молитвах, религиозных иерархиях, священных символах и практиках такных культов;
* _animal_care: int - (Мудрость) навык отвечает за следующие проверки: возможность успокоить одомашненное животное, удержать скакуна от паники, почувствовать намерения животного;
* _insight: int - (Мудрость) навык отвечает за следующие проверки: определение истинных намерений существа, распознавание лжи, предсказание следующего шага;
* _medicine: int - (Мудрость) навык отвечает за следующие проверки: лечение ран и болезней, стабилизация умирающего спутника, диагностирование болезней, определение причины смерти;
* _perception: int - (Мудрость) навык отвечает за следующие проверки: возможность увидеть/услышать/унюхать/ощутить чьё-либо присутствие, готовность к новым событиям и остроту чувств;
* _survival: int - (Мудрость) навык отвечает за следующие проверки: выслеживание врагов, охота на дичь, нахождение пути, определение опасности поблизости, предсказание погоды, избегание природных опасностей;
* _deception: int - (Харизма) навык отвечает за следующие проверки: способность утаить правду (речами или действиями), ложь, введение в заблуждение;
* _intimidation: int - (Харизма) навык отвечает за следующие проверки: угрозы, враждебные действия и физическое насилие;
* _performance: int - (Харизма) навык отвечает за следующие проверки: удовлетворение публики музыкой, танцем, актёрской игрой, рассказом и т.д.;
* _persuasion: int - (Харизма) навык отвечает за следующие проверки: влияние на других тактично, с уважением и добродушием.
